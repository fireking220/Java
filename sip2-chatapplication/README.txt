This program builds a chat application complete with GUI's for the client including a log in screen, register screen, and chat window. A client will be able to log into an account, register for a new account, log into a specified server, and send messages to everyone currently connected to that server. The server and client will be listening for incoming data and specified sockets, each assigned its own thread, and react accodingly to the data sent. Also detects when clients log in or out and updates the online list accordingly. Will also keep a log of the entire chat hostory in a text file.
